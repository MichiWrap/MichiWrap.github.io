Toy Escape

Endless Slingshot Arcade Game

2015-2016



Toy Escape is an endless arcade game on itunes. It uses a unique slingshot-style mechanic to propel the character through upcoming platforms. The user can choose from and play as a myriad of slinky toys. Each may play the same, but they all have unique effects and quirks to add flavor to the game. The objective is to climb as many floors as possible in an attempt to escape the toy factory.

Roles and Responsibilities

Character Design


The Challenge: To create characters that are unique, that can draw user interest, and--most importantly--that can share a single set of animations.


The Solution: I have designed "slinky toy" characters with a base structure. This way their details can be drastically varied in order to create unique designs, but they can still share a single animation set.

Environment Design


The Challenge: To create an environment that matches our character concepts and that can be designed in away to allow parallax scrolling.


The Solution: I designed a factory-esque background using SVG assets. In this design, I can be relatable and create many separate assets using efficient space from the SVG format. These assets could be layered and modularly placed to form a coherent background that can smoothly parallax.

UI Design


Among character and environment designs, I was also in charge of creating UI and other UX pages. I created flat, clear UI designs using complimenting colors and silhouetted images to define button purposes while still being relatable to Toy Escape in terms of style. I also designed UX pages, such as the prize pages and character dispenser that can be found in game.

Advertising


After production, I created a few pieces for Toy Escape's promotion. These include screenshots, videos, and still and gif images.
